# Sistem Informasi Akuntansi

Sistem Informasi Akuntasi merupakan sistem berbasis website yang dapat digunakan pengguna untuk mempermudah proses pencatatan transaksi keuangan.

# Feature
- Login / Register
- Dashboard
- Transaction (CRUD)
- Daily Report
- Search and Filter
- Responsive UI

# Teknologi yang Digunakan
- ReactJS
- Vite
- Tailwind

# Instalasi
npm install

# Menjalankan Project (Development)
npm run dev

# Buat .env.local
VITE_API_URL=http://localhost:{portBackend}/api
